# Project Summary
The Sistema PDV is a comprehensive Point of Sale (POS) and inventory management system developed in Python, utilizing a Tkinter graphical interface and SQLite database. Specifically designed for clothing retail businesses, it offers features for managing products, clients, sales, and inventory movement. The system aims to streamline operations for small to medium-sized retailers, enhancing efficiency and customer service.

# Project Module Description
## Key Functionalities
1. **Product Management**: Complete registration with advanced search and filtering capabilities.
2. **Sales System (POS)**: Intuitive interface for managing sales, including customer linkage and multiple payment options.
3. **Client Management**: Comprehensive client registration with purchase history tracking.
4. **Inventory Movement**: Records entries and exits, maintaining detailed movement history.
5. **Update System**: Automatic version checking and updates via GitHub.

# Usage
### Installation
1. **Prerequisites**:
   - Python 3.8 or higher
   - Libraries: tkinter (usually included), requests

2. **Install dependencies**:
   ```bash
   pip install requests
   ```

3. **Run the system**:
   ```bash
   python main.py
   ```

### First Use
- The system will automatically create the `estoque.db` database on the first run.
- Database location:
  - **Windows**: `%LOCALAPPDATA%\EstoqueLoja\estoque.db`
  - **Linux/Mac**: `~/.local/share/estoque_loja/estoque.db`

### Recommended Workflow
1. Register Products
2. Record Inventory Entries
3. Perform Sales
4. Register Clients
